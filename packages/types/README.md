# @wflow-analyzer/types

Shared TypeScript types for the n8n workflow analyzer.

```ts
import type { AnalysisReport, Violation, N8nWorkflow, GraphAnalysis } from "@wflow-analyzer/types";
```

Used internally by `@wflow-analyzer/core` and `@wflow-analyzer/server`. Install this directly if you are consuming analyzer reports in your own TypeScript project.
